# Raymon – Frontend Coding Task

این پروژه در پاسخ به یک تسکت از شرکت **رایمون** پیاده‌سازی شده است.  
هدف پروژه دریافت لیست کاربران از یک REST API و نمایش آن‌ها به صورت ریسپانسیو با استفاده از HTML، Tailwind CSS و JavaScript (Vanilla) است.

---

##  هدف پروژه
- HTML & CSS (با Tailwind CSS)
- JavaScript (Vanilla)
- مصرف REST API
- مدیریت State (Loading / Error)
- طراحی Responsive
- کار با Git و GitHub
- کدنویسی تمیز و قابل نگهداری

---

##  API مورد استفاده
داده‌ها از API زیر دریافت می‌شوند:

```
https://jsonplaceholder.typicode.com/users
```

---

##  ویژگی‌ها
- دریافت داده‌ها با `fetch`
- نمایش کاربران به صورت ** Card**
- نمایش اطلاعات:
  - Name
  - Email
  - City
- نمایش وضعیت **Loading** هنگام دریافت داده
- طراحی کاملاً **Responsive**
- استایل‌دهی با **Tailwind CSS**

---

##  تکنولوژی‌های استفاده‌شده
- HTML
- Tailwind CSS
- JavaScript (Vanilla)
